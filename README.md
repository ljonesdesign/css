# CSS Cascading Style Sheets

## CSS affects html Elements

```
Selector
	|
	p { 
	    font-family: Arial;}                 
	    |_________________|
	             |
	       declaration
             	       
	    font-family: Arial;
	    |___________|_____|
	          |        |  
	     property:   value;
	          |        |
	        color:  yellow;
	        
	  p {
	     font-family: sans-serif;
	    }     
     
```
The CSS selector ```p``` will affect paragraphs in HTML.
